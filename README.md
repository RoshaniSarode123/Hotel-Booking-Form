# Hotel-Booking-Form
Hotel Booking Form is a responsive web-based form developed using HTML5, CSS3, and JavaScript.
The form allows users to select check-in/check-out dates, room type, guest count, and add special requests.
It includes proper date validation to ensure the check-out date is later than the check-in date and prevents past date bookings.
✨ Features:
Check-in & Check-out Date Picker
Date Validation (Check-out > Check-in)
Room Type Selection (Single, Double, Suite)
Guest Count Input
Special Requests Textarea
Responsive Design (Mobile, Tablet, Desktop)
Dynamic Error Messages
🛠 Technologies Used:
HTML5
CSS3
JavaScript
